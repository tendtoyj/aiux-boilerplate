## 1. 잠재 사용자 대상 사전조사

**목적**: 페인포인트·니즈 파악 / 경쟁 제품 사용 현황 / 컨셉 초기 반응 / 행동 패턴 이해

**주요 조사 내용**
- 현재 겪고 있는 문제점·불편사항
- 기존 대안 솔루션 사용 경험 및 만족도
- 의사결정 과정 및 구매·사용 동기
- 인구통계학적 정보 및 라이프스타일

**특징**: 열린 질문(오픈 엔드) 탐색 / 경쟁 제품 비교 / 가격 민감도 / 컨셉 관심도

---

## 2. 기존 사용자 대상 조사

**목적**: 사용 경험 기반 개선점 도출 / 사용성 평가 / 사용 패턴 / 충성도·NPS

**주요 조사 내용**
- 기능별 만족도 및 사용 빈도
- 사용 과정 어려움·불편사항
- 서비스 사용 패턴 및 활용 시나리오
- NPS, 만족도, 개선 요청

**특징**: 실제 경험 기반 구체적 질문 / 정량+정성 병행 / 사용 로그 연계 가능

---

## 3. 브랜드/마케팅 인지 관련 조사

**목적**: 인지도·이미지 파악 / 캠페인 효과 / 포지셔닝 확인 / 인식 변화 추적

**주요 조사 내용**
- 비보조·보조 인지도
- 브랜드 연상 이미지 및 속성
- 광고·마케팅 기억 및 영향력
- 브랜드 충성도·선호도

**특징**: 전후 비교 설계 / 세멘틱 차이 척도 / 경쟁 브랜드 비교 / 잠재·기존 고객 인식 차이

---

## 4. 신규 기능 니즈 수집 조사

**목적**: 관심도 측정 / 우선순위 식별 / 요구사항 상세 파악 / 가치 제안 검증

**주요 조사 내용**
- 현재 불편사항·개선 요구
- 신규 기능 중요도·관심도 평가
- 상세 요구사항, 지불 의향
- 경쟁 제품 유사 기능 경험

**특징**: 명확한 컨셉 설명 / 순위 매기기 / 프로토타입·목업 활용 / 시나리오 기반 질문

---

## 5. 사용자 인터뷰 대상자 모집 조사

**목적**: 적합 프로필 확인 / 다양한 세그먼트 표본 / 참여 의향·일정 / 사전 정보

**주요 조사 내용**
- 기본 인구통계
- 사용 경험·빈도, 특정 경험 여부
- 참여 가능 일정·선호 방식
- 연락처

**특징**: 간결·짧은 설문 / 명확한 스크리닝 / 개인정보 동의 필수 / 인센티브 안내

---

## 6. 특정 프로모션/이벤트 관련 조사

**목적**: 참여 경험 평가 / 효과·만족도 / 개선점 / 인식 변화 확인

**주요 조사 내용**
- 인지 경로, 참여 동기, 관련 사용 경험
- 만족도·개선점, 향후 참여 의향

**특징**: 참여자 타겟팅 / 직후 신속 진행 / 경품 평가 / 브랜드 인식 영향 측정

---

## 7. 제품/서비스 중단 원인 조사

**목적**: 중단 이유 파악 / 불만족 요소·이탈 트리거 / 경쟁사 전환 분석 / 복귀 조건

**주요 조사 내용**
- 중단 시점·계기, 불만족 요소
- 대체 솔루션 전환 여부·이유
- 재사용 의향·필요 조건

**특징**: 이탈 시점별 세그먼트 / 결정적 순간 집중 / 익명성 보장 / 재유치 대상 식별

---

## 8. 경쟁사 비교 분석 조사

**목적**: 강점·약점 파악 / 차별화 요소 / 사용자 관점 비교 / 개선 우선순위

**주요 조사 내용**
- 경쟁사 사용 경험, 기능별 만족도 비교
- 가격 대비 가치, 차별화 장단점
- 복수 서비스 사용자 선호도

**특징**: 블라인드 테스트 가능 / 비교 매트릭스 / 전환 의향·조건 / 정기 추적

---

## 9. 가격 정책 수립을 위한 조사

**목적**: 적정 가격대 / 민감도·탄력성 / 가격 구조 선호도 / 가격 대비 가치

**주요 조사 내용**
- 최대·최소 지불 의향, 가격 모델 선호
- 가격 변동에 따른 사용 의향 변화
- 경쟁사 가격 비교 인식

**특징**: Van Westendorp / Gabor-Granger / 컨조인트 분석 / 세그먼트별 차이

---

## 10. 고객 세그먼테이션 조사

**목적**: 핵심 그룹 식별 / 그룹별 니즈·행동 / 맞춤형 전략 / 성장 세그먼트

**주요 조사 내용**
- 심층 인구통계, 사용 목적·가치
- 사용 행태·습관, 구매 과정·영향 요인
- 라이프스타일·가치관

**특징**: 클러스터 분석 / 정량+정성 결합 / 다각적 세그먼테이션 / 페르소나 연계

## 11. 사용자 여정 맵 작성을 위한 조사

**목적**: 전 과정 경험 흐름 / 터치포인트 만족·불만족 / 병목·개선 기회 / 여정 시각화

**주요 조사 내용**
- 인지~지속 사용 단계별 경험·감정
- 주요 결정 지점·영향 요인
- 채널 전환, 포기·이탈 지점

**특징**: 사용자 일기 방식 / 터치포인트별 상세 / 정량+정성 / 관찰 연구 연계

## 12. 제품/서비스 컨셉 테스트

**목적**: 출시 전 초기 반응 / 강점·약점 / 관심도·사용 의향 / 메시지·가치 제안 검증

**주요 조사 내용**
- 이해도·초기 인상, 가치 제안 공감도
- 예상 사용 빈도·상황, 구매 의향
- 차별성·개선 제안

**특징**: 목업·프로토타입 활용 / A/B 테스트 / 정성+정량 / 세그먼트별 반응

## 13. 사용자 성향 및 행동 패턴 조사

**주요 조사 내용**

- 의사결정 과정, 디지털 활용 성향
- 가치관·라이프스타일 태도

**특징**: 행동 관찰+설문 복합 / 심리학 척도 / 장기 추적 / 심층 인터뷰 연계

## 14. 사용자 만족도 정기 트래킹 조사

- 경쟁사 대비 상대적 변화

**특징**: 동일 지표 정기 측정 / 간결한 설문 / 패널 추적 / 정량+정성 균형

## 19. 내부 임직원 대상 제품/서비스 평가

**주요 조사 내용**

- 내부 시스템 사용성, 고객 니즈 충족 제안
- 경쟁사 대비 인식

**특징**: 부서·직무별 차별 설문 / 익명성 / 정기 내부 평가 / 아이디어 제안 프로세스
