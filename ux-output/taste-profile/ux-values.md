# UX Values — 입맛 프로필 (Taste Profile)

> Feature: 입맛 프로필 — 기존 헤비 유저의 예약·탐색·리뷰 등 축적된 데이터를 AI가 분석하여, 사용자 자신도 몰랐던 식사 취향을 시각적으로 보여주고 맞춤 추천으로 연결하는 기능

## Goal

- Match Table은 맛집·카페를 큐레이션하고 예약까지 연결하는 서비스. 탐색 → 큐레이션 → 예약이 핵심 플로우
- 입맛 프로필의 타겟은 **이미 데이터가 충분히 쌓인 기존 장기 유저**. 신규 유저가 아니라 예약 내역, 저장 목록, 리뷰, 검색 기록 등이 풍부한 사람들
- 이 유저들은 서비스를 꾸준히 쓰고 있지만 방문 빈도가 점차 줄고 있고, 경쟁사도 많아지는 상황. 입맛 프로필이라는 새로운 경험을 통해 다시 서비스에 대한 관심을 끌어올리고, 새로운 탐색과 예약으로 이어지게 만드는 것이 핵심 목표
- 비즈니스 성과 지표는 **예약 전환율**. 경영진이 AI 기능 도입을 강하게 드라이브하고 있으며, 입맛 프로필은 AI를 처음 투입하는 피처로 사내 관심도가 높음

## UX Values

1. **"나 이런 사람이었어?" — 데이터가 알려주는 나도 몰랐던 내 입맛**
    - MBTI나 성격 테스트를 사람들이 좋아하는 이유는 "나에 대한 새로운 해석"을 받아보는 재미가 있기 때문. 입맛 프로필은 여기서 한 단계 더 나아감 — 주관적 설문이 아니라 실제 예약 내역, 리뷰, 저장 목록 같은 행동 데이터에 기반하기 때문에 "AI가 내 데이터를 분석해줬다"는 신빙성이 붙음.
    - ChatGPT에서 "지금까지 우리 대화를 바탕으로 나를 분석해줘"라는 프롬프트가 크게 유행한 것처럼, 사람들은 자기 데이터를 기반으로 한 분석 결과에 강한 흥미를 느낌. 특히 "매운맛 러버", "숨은 카페 탐험가" 같이 나를 한 마디로 정의해주는 프레이밍이 있으면 더 몰입하게 됨.
    - 이 순간에 느끼는 감정은 "와, 되게 신기하다"에 가까움. 내가 매일 하던 식사와 예약이 하나의 프로필로 정리되어 눈앞에 보이는 경험. 단순히 통계 숫자를 나열하는 것이 아니라 "당신은 이런 사람이에요"라는 서사가 있어야 함.
    - 참고: 스포티파이 Wrapped가 매년 폭발적 반응을 얻는 것도 같은 맥락 — 내 데이터를 재해석해서 "올해 당신은 이런 청취자였어요"라고 스토리로 돌려주니까 재밌는 것.

2. **"맞아, 이건 진짜 내 얘기네" — 분석 결과에 고개를 끄덕이는 공감**
    - 결과를 보면서 "아, 맞아 나 확실히 이런 쪽이지"라고 스스로 납득하는 순간. 이건 정확도의 문제이기도 하지만, 사용자의 실제 경험과 연결되는 구체성이 핵심.
    - "한식을 좋아합니다"는 너무 뻔해서 감흥이 없고, "금요일 저녁에는 유독 일식을 예약하는 경향이 있어요"처럼 본인도 의식하지 못했지만 돌이켜보면 맞는 패턴이 나와야 "오, 진짜네?"라는 반응이 나옴.
    - 반대로 "내가 이렇다고?" 하면서 의외성을 느끼는 것도 좋은 반응. 공감과 의외성 모두 사용자를 결과에 몰입하게 만드는 힘이 있음. 중요한 건 "틀린 느낌"이 아니라 "생각해보니 그럴 수도 있겠다"는 수긍 가능한 의외성이어야 한다는 것.
    - 참고: 타로카드나 운세가 "맞다"고 느끼는 건 바넘 효과이기도 하지만, 입맛 프로필은 실제 데이터 기반이므로 더 강한 납득감을 줄 수 있음. 다만 너무 뻔한 결과는 오히려 실망을 주므로, "알고 있던 것의 확인"과 "몰랐던 패턴의 발견"이 적절히 섞여야 함.

3. **"이거 인스타에 올려야지" — 나만의 결과물을 공유하고 싶은 욕구**
    - 분석 결과가 흥미로우면 자연스럽게 다른 사람에게 보여주고 싶어짐. SNS나 인스타그램에 올리면 "나도 해봐야지"라는 반응이 생기고, 이는 서비스 바이럴로 이어질 수 있음.
    - 이때 결과물의 형태가 중요함. 텍스트 분석 리포트를 캡처해서 올리는 것과, 시각적으로 예쁘고 한눈에 파악되는 카드형 이미지를 공유하는 건 전혀 다른 경험. 공유하고 싶어지려면 "이걸 보는 사람이 궁금해할 만한" 비주얼이어야 함.
    - 참고: 스포티파이 Wrapped, MBTI 결과 카드, 카카오 연말정산 같은 것들이 매번 SNS를 도배하는 이유는 결과물 자체가 공유에 최적화된 포맷이기 때문. "내 결과는 이건데, 너는?" 하는 비교 욕구가 자연스럽게 생기는 구조.

4. **"이건 진짜 나만의 그림이다" — AI가 생성한 나만의 비주얼**
    - 한동안 AI에게 "우리 대화를 바탕으로 그림을 그려줘"라는 요청이 크게 유행한 것처럼, 텍스트 분석을 넘어서 내 식사 데이터를 기반으로 AI가 맞춤형 이미지를 생성해주는 경험. 사람마다 데이터가 다르니까 결과물도 전부 다르고, 그래서 "이건 세상에 나만 가진 것"이라는 특별함이 생김.
    - 이 유니크한 비주얼은 공유 욕구를 극대화함. 텍스트로 "당신은 매운맛 러버입니다"라고 보여주는 것과, 내 입맛 데이터가 반영된 독특한 일러스트가 생성되는 것은 감정적 임팩트가 완전히 다름. "내 친구 거는 어떻게 생겼을까?" 하는 궁금증이 자연스럽게 바이럴을 만듦.
    - 참고: ChatGPT의 "우리 관계를 그림으로 그려줘" 트렌드, 지브리 스타일 변환 유행 등 — 나의 데이터가 시각적 결과물로 변환되는 경험에 사람들이 강하게 반응한다는 검증된 패턴.

5. **"여기는 내 취향이니까 한번 가볼까" — 분석 위에 얹어지는 추천의 신뢰감**
    - 일반적인 "인기 맛집 추천"과 "당신의 입맛 프로필 기반 추천"은 신뢰 수준이 다름. 내 데이터를 분석한 결과 위에 추천이 얹어지면 "이건 진짜 나한테 맞는 곳이겠다"는 기대가 생기고, 그 기대가 새로운 장소를 탐색하고 예약까지 이어지는 동기가 됨.
    - "매운맛 러버인 당신에게 이런 곳을 추천드려요"처럼 프로필과 추천이 연결되어 있다는 걸 명시적으로 보여주는 게 중요. 왜 이 식당이 나한테 추천됐는지 맥락이 보이면 신뢰감이 더 올라감.
    - 이 부분이 결국 예약 전환율이라는 비즈니스 성과와 직결되는 지점. 입맛 프로필 자체의 재미로 유저를 끌어들이고, 그 위에 자연스럽게 "그래서 여기 가보세요"가 얹어지는 흐름.
    - 참고: 넷플릭스의 "회원님이 좋아할 만한 콘텐츠" 추천은 그냥 Top 10보다 클릭률이 높음. 개인화되었다는 인식 자체가 탐색 의지를 높이는 효과.

6. **(사이드) "옛날에도 맥모닝파였네" — 시간에 따른 취향 변천사**
    - 1년 전의 나와 지금의 나, 계절마다 달라지는 패턴 같은 걸 데이터로 보여주는 경험. "예전에는 강남 위주로 예약했는데 요즘은 홍대가 많아졌네", "여름에는 확실히 냉면 예약이 늘어나네" 같은 발견.
    - 다만 이게 의미 있으려면 데이터가 충분히 풍부해야 해서, 서비스 초기부터 메인으로 가져가기는 어려움. 장기적으로 데이터가 쌓인 유저에게 점진적으로 제공하는 방향이 현실적.
    - 참고: 애플 "올해의 사진" 회고, 인스타그램 연말 하이라이트처럼 시간 축 위에서 자기 데이터를 되돌아보는 경험은 감성적 만족도가 높음.

7. **(사이드) "우리 입맛 일치율 87%래!" — 친구와 비교하는 재미**
    - 친구나 연인과 입맛 프로필을 비교해서 일치율을 보여주거나, 함께 갈 식당을 추천받는 경험. "같이 넣어가지고 우리 이 식당에 대한 입맛 일치율"을 분석해주는 방식도 가능.
    - 현재 서비스에서도 초대 기능이나 "여기어때" 같은 공유 기능이 있으므로 자연스럽게 연결될 수 있는 확장 포인트.
    - 다만 메인 피처가 되기보다는, 개인 입맛 프로필이 안착한 이후에 고도화하는 방향이 적절.
    - 참고: 스포티파이 Blend(친구와 음악 취향 비교), 카카오 궁합 테스트 등 — 나와 상대방을 비교하는 포맷은 참여율과 공유율이 높음.
