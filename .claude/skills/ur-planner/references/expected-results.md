# 예상 결과 참고 예시

이 문서는 조사 계획서의 "예상 결과" 섹션 작성 시 참고할 수 있는 구체적인 예시를 제공합니다. 조사 유형과 맥락에 맞게 적절히 변형하여 활용하세요.

---

## 예시 1: 온라인 쇼핑 장바구니 이탈 분석

### 기술통계

- **응답자 프로필:** 총 1000명 (응답률 85%), 성별/연령대/거주지역/쇼핑빈도/금액 분포
- **구매 포기 이유 분포 (복수응답):**
  - 가격 요인: 예상보다 높은 총 비용 (58%), 더 저렴한 가격 탐색 (42%)
  - 배송 요인: 배송비 부담 (55%), 긴 배송 시간 (25%)
  - 상품 관련: 정보/품질 불확실 (30%), 부정적 리뷰 (28%)
  - 결제 과정: 복잡한 절차 (22%), 원하는 결제수단 부재 (12%)
  - 외부 요인: 다른 상품 발견 (35%), 충동구매 자제 (30%)

### 교차분석 인사이트

- **성별 차이:** 여성이 배송비/총 비용에 더 민감, 남성은 결제 절차 불편에 민감
- **연령대별:** 20-30대는 가격 탐색/리뷰에 민감, 50대 이상은 복잡한 절차/기술적 오류에 높은 포기율
- **쇼핑 빈도별:** 헤비 유저는 가격 비교/배송 조건에 민감, 라이트 유저는 변심/잊어버림 비율 높음

### 세그먼트 프로파일링

- **세그먼트 A (가격 민감형 알뜰 쇼핑족):** 20-30대 여성, 중하위 금액권, 할인/쿠폰에 가장 효과적 반응
- **세그먼트 B (편의성 중시형 계획 구매자):** 30-40대 직장인, 간편결제/빠른 배송 시 전환율 상승
- **세그먼트 C (신중 탐색형 잠재 고객):** 전 연령 분포, 재고/가격 변동 알림, 리마인더가 유효

## 예시 2: 브랜드/마케팅 인지도 조사 (메신저 앱)

### 기술통계

- **브랜드 인지도:** 비보조 인지도 75%, 보조 인지도 95%
- **사용 빈도:** 활성 사용자(월 15회 이상) 55%, 비활성 30%, 인지만 10%
- **인지 경로:** 지인 추천 45%, TV/온라인 광고 30%, 앱스토어 15%
- **브랜드 연상 키워드:** 편리함 65%, 친근함 55%, 대중적 48%
- **NPS:** +45 (추천자 65% - 비추천자 20%)

### 교차분석 인사이트

- **성별:** 여성이 비보조 인지도 높음 (78% vs 72%), 브랜드 충성도 0.4점 높음
- **연령대:** 20대(85%)와 30대(82%)가 가장 높은 인지도, 50대 이상 60%로 최저
- **사용 빈도:** 헤비유저 비보조 인지도 92%, 라이트유저 55%

### 경쟁사 비교

- 사용 편의성에서 우위, 혁신성/기능 다양성에서 열위
- 차별적 강점: 사용 편의성(55%), 대중성(48%), 안정성(42%)

### 세그먼트 (5개)

- 충성 사용자 (30%): 20-30대, 매일 사용, 강한 충성도
- 실용형 사용자 (25%): 30-40대, 업무 목적, 기능성 중시
- 사회적 사용자 (20%): 10-20대, 그룹 소통 중심
- 간헐적 사용자 (15%): 다양한 연령, 경쟁 서비스 병행
- 잠재적 사용자 (10%): 40-50대, 디지털 활용도 낮음

## 예시 3: 배달앱 신규 기능 니즈 조사

### 기술통계

- **응답자 프로필:** 총 1000명 (응답률 80%)
- **주문 포기 이유 분포:**
  - 가격 요인: 높은 배달비 (65%), 최소 주문 금액 부담 (40%)
  - 배달 요인: 예상 배달 시간 너무 김 (35%), 배달비 정책 불만 (30%)
  - 메뉴/식당 관련: 원하는 식당/메뉴 없음 (50%), 메뉴 정보/리뷰 부족 (25%)
  - 결제/주문 과정: 주문 단계 복잡 (15%), 결제 수단 부재 (10%)
  - 외부 요인: 다른 앱에서 더 좋은 조건 발견 (30%)

### 교차분석 인사이트

- **성별:** 여성이 배달비/최소 주문 금액에 더 민감, 남성은 배달 시간/앱 성능에 민감
- **연령대:** 20-30대는 경쟁 앱 비교/할인에 민감, 40대 이상은 메뉴 부족/앱 사용 어려움
- **이용 빈도:** 헤비 유저는 서비스 디테일에 민감, 라이트 유저는 기본 선택폭 문제
- **이용 금액:** 고액 이용자는 서비스 품질에 민감, 소액 이용자는 절대적 비용 부담

## 활용 가이드

예상 결과를 작성할 때:

1. **연구질문에 대한 답변** 형태로 구성
2. **구체적인 수치**를 포함하여 손에 잡히는 예시 제시
3. **교차분석 인사이트**를 통해 세그먼트별 차이 강조
4. **세그먼트 프로파일링**으로 타겟별 맞춤 전략 연결
5. **인터뷰의 경우:** 사용자 인사이트 → 추후 액션 연결 형식
6. **설문의 경우:** 얻고 싶은 결과에 따른 문항 구성 역설계 고려
