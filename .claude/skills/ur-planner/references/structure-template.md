| 조사 유형 | 추가 고려 항목 |
|-----------|----------------|
| 잠재 사용자 사전조사 | 컨셉 테스트 방법론, 경쟁 분석 프레임워크, 가격 민감도 테스트 |
| 기존 사용자 만족도 | NPS/CSAT/CES 측정, 기능별 사용성 평가, 이탈 위험 사용자 식별 |
| 장바구니 포기 원인 | 구매 여정 맵핑, 전환율 개선 가설 테스트, 마찰 지점 식별 |
| 브랜드/마케팅 인지도 | 보조/비보조 인지도 측정, 브랜드 연상 맵핑, 캠페인 효과 측정 |
| 신규 기능 니즈 수집 | 기능 우선순위 결정, 컨셉 시각화, 사용 시나리오 기반 피드백 |
| 사용자 여정 맵 작성 | 여정 단계 정의, 터치포인트별 감정 측정, 여정 시각화 |
| 가격 정책 수립 | Van Westendorp/Gabor-Granger, 가격-가치 인식 맵핑 |
| 제품/서비스 중단 원인 | 이탈 단계 식별, 재유입 가능성 평가, 경쟁사 전환 패턴 |
| 경쟁사 비교 분석 | 경쟁 요소 우선순위, 블라인드 테스트, 포지셔닝 맵 |
| 고객 세그멘테이션 | 세그먼트 변수 선정, 클러스터링 방법론, 페르소나 개발 |
| 로열티 프로그램 효과성 | 로열티 지표 정의, 혜택 요소 평가, ROI 계산 모델 |
| 채널별 사용자 경험 | 옴니채널 경험 맵핑, 채널 선호도 결정 요인, 채널별 최적화 |
